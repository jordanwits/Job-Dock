  ? Want to override settings? No

ADD CUSTOM DOMAIN (OPTIONAL)
================================================================================

Want to use your own domain (e.g., app.yourdomain.com)?

1. In Vercel Dashboard:
   Project ‚Üí Settings ‚Üí Domains ‚Üí Add Domain

2. Enter: app.yourdomain.com

3. Add DNS records (Vercel shows you exactly what to add):
   Type: CNAME
   Name: app
   Value: cname.vercel-dns.com

4. Wait for DNS propagation (5-30 minutes)

5. Vercel automatically provisions SSL!

6. Update infrastructure/config.ts:
   vercelDomain: 'app.yourdomain.com',

7. Redeploy AWS:
   cd infrastructure
   npm run deploy:prod

Done! No ACM certificates needed - Vercel handles it all!

================================================================================
TROUBLESHOOTING
================================================================================

"CORS error":
  ‚Üí Make sure you updated infrastructure/config.ts with your Vercel URL
  ‚Üí Make sure you redeployed AWS: cd infrastructure; npm run deploy:prod
  ‚Üí Clear browser cache

"Can't log in":
  ‚Üí Check Cognito callback URLs include your Vercel domain
  ‚Üí Verify environment variables in Vercel
  ‚Üí Check browser console for errors

Build fails on Vercel:
  ‚Üí Check build logs in Vercel dashboard
  ‚Üí Verify package.json has all dependencies
  ‚Üí Make sure vercel.json is correct

"Environment variables not working":
  ‚Üí Must redeploy after adding env vars
  ‚Üí Check they're set for "Production" environment
  ‚Üí Verify names start with VITE_ prefix
