STEP 1: Deploy to Vercel
-------------------------

Choose ONE method:

METHOD A - Dashboard (Easiest):
  1. Go to: https://vercel.com/new
  2. Import your Git repository
  3. Add environment variables (see list below)
  4. Click "Deploy"
  5. Get your URL: https://jobdock-xxxx.vercel.app

METHOD B - CLI (Faster):
  1. npm install -g vercel
  2. vercel
  3. Add env vars: vercel env add (for each variable)
  4. vercel --prod
  5. Get your URL: https://jobdock-xxxx.vercel.app


STEP 2: Update AWS with Your Vercel URL
----------------------------------------

After you get your Vercel URL, update infrastructure/config.ts:

  prod: {
    vercelDomain: 'jobdock-xxxx.vercel.app',  // <-- YOUR ACTUAL VERCEL URL
    ...
  }

Then redeploy AWS (takes 1-2 minutes):

PowerShell:
