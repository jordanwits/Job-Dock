================================================================================
HOW TO DEPLOY (3 SIMPLE STEPS)
================================================================================

STEP 1: Validate Prerequisites
-------------------------------
npm run validate:prereqs

This checks:
  - AWS CLI installed and configured
  - Node.js and npm installed
  - Project dependencies installed
  - AWS credentials valid
  - SES configuration status

Fix any issues before proceeding.


STEP 2: Choose Your Deployment Path
------------------------------------

PATH A - WITHOUT Custom Domain (Fastest):
  npm run deploy:prod
  
  When prompted:
    - Press Enter to skip domain
    - Press Enter to skip certificate
    - Type 'y' to confirm deployment
  
  Time: ~15-20 minutes
  Result: App available at CloudFront URL


PATH B - WITH Custom Domain:
  1. Create ACM certificate in AWS (5-10 min)
  2. Edit infrastructure/config.ts:
     - Set domain: 'app.yourdomain.com'
     - Set cloudfrontCertificateArn: 'arn:...'
  3. Run: npm run deploy:prod
  4. Add CNAME to your DNS provider (instructions shown)
  
  Time: ~20-30 minutes
  Result: App available at https://app.yourdomain.com


STEP 3: Verify Deployment
--------------------------
npm run check:prod

Then test:
  - Visit the URL
  - Create an account
  - Test app features
