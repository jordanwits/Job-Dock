# Saved line items

**Route:** `/app/line-items`

Catalog of reusable descriptions and default pricing for quotes and invoices so you do not retype common work items.

## Who can use it

Typically **owners and admins** behind the same access as other admin areas (not on employee sidebar).

## Workflow

1. Open **Saved line items**.  
2. Create or import rows (CSV import exists in some builds for bulk catalog maintenance).  
3. When building a quote or invoice, pull from saved items where the UI offers catalog selection or paste flows.

## Troubleshooting

- **Import errors** — Validate CSV columns match expected headers; check currency/number formats.  
- **Item not appearing on new quote** — Confirm save succeeded; refresh Quotes page.
