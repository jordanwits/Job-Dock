NEXT STEPS - WHAT YOU NEED TO DO
================================================================================

OPTION 1: Deploy WITHOUT Custom Domain (Fastest)
-------------------------------------------------

Just run:
  npm run deploy:prod

When prompted:
  - Press Enter to skip domain
  - Press Enter to skip certificate
  - Type 'y' to confirm AWS deployment
  - Type 'y' to confirm SES setup

Your app will be available at the CloudFront URL (shown at end).
Time: ~15-20 minutes

You can add a custom domain later!


OPTION 2: Deploy WITH Custom Domain
------------------------------------

Pre-requisites:
  1. Create ACM certificate (5-10 min):
     - Go to: https://console.aws.amazon.com/acm/home?region=us-east-1
     - Request certificate for your domain
     - Add DNS validation records
     - Wait for "Issued" status
     - Copy the ARN

  2. Update infrastructure/config.ts:
     - Uncomment and set 'domain' field
     - Uncomment and set 'cloudfrontCertificateArn' field
     - Save file

  3. Run deployment:
     npm run deploy:prod

  4. Add DNS CNAME (instructions will be shown):
     - Add CNAME record to your DNS provider
     - Point to CloudFront distribution
     - Wait 5-30 minutes for propagation

Time: ~20-30 minutes (including ACM certificate wait time)
